технологій Microsoft SQL Server ".

Опис проекту
Проект являє собою повноцінну Business Intelligence систему для аналізу даних про співробітників та їхні відпустки. Система автоматизує шлях даних від операційної бази (OLTP) до інтерактивного дашборду.

Технологічний стек
БД: SQL Server 2022 (Сховище за схемою "Зірка").
Генерація даних:** Redgate SQL Data Generator 
ETL: SQL Server Integration Services (SSIS).
OLAP: QL Server Analysis Services (SSAS Multidimensional Cube).
Reporting:SQL Server Reporting Services (SSRS).

Структура репозиторію
/Databases` — SQL-скрипти створення таблиць, зв'язків та файл проекту Redgate.
/HR_ETL_Project` — SSIS-пакети для очищення та завантаження даних у сховище.
/CW_OLAP` — багатовимірний куб з налаштованими вимірами, ієрархіями та KPI.
/Reports` — SSRS звіти (Матричні, Діаграми, Drill-down та аналітичний Dashboard).

Oсновні досягнення
Автоматизація: Реалізовано повний ETL-цикл з логуванням та обробкою помилок.
Аналітика: Розраховано складні метрики (Moving Average, YTD, Rank) за допомогою MDX.
Візуалізація: Побудовано інтерактивні звіти з Document Map, параметрами мульти-вибору та умовним форматуванням.
